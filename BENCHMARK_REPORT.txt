HAMT BENCHMARK REPORT
=====================
Date: 2026-04-11
Status: Neuromorphic experiments in progress


EXECUTIVE SUMMARY

HAMT (Habituation-Aware Metabolic Training) reduces spiking neural
network inference energy by 15-41% on pattern recognition tasks
(digits, characters, fashion) with no accuracy loss. Performance on
natural image classification (CIFAR, SVHN) is weak or negative.
Neuromorphic event-driven dataset experiments are in progress and
expected to show stronger results due to genuine temporal patterns.


METHODOLOGY

Config (verified optimal):
  lambda_energy = 0.001
  lambda_habituation = 0.0005
  target_spike_rate = 0.05
  ramp_epochs = 10
  hidden_size = 800 (FF) or Conv16-Conv32-FC
  num_steps = 25
  epochs = 15
  seed = 42

Energy model: Loihi 2 estimates (120 pJ/spike, 23 pJ/synop)

NOTE: Aggressive config (0.01/0.005, h=400) was tested and produces
only 0-9% savings. Default config is strictly superior. See
feedback_hamt_hyperparams.md in project memory.

KEY FINDINGS
============

1. HAMT works best on feedforward architectures with h=800.
   Average FF energy reduction: 21.1%
   Average Conv energy reduction: 8.9% (excluding SVHN)

2. HAMT works best on pattern recognition tasks (digits, characters,
   fashion items) where input patterns repeat predictably.

3. Natural image classification (CIFAR, SVHN) shows weak or negative
   results. These datasets have high inter-class visual diversity and
   low intra-class repetition, giving habituation little to learn.

4. Fashion-MNIST shows the widest range (5.8% conv to 40.7% FF),
   suggesting the FF architecture allows habituation to be more
   selective about which spikes to suppress.

5. HAMT never significantly hurts accuracy. Average accuracy delta
   across all valid experiments is +0.28% (slightly positive).


COMPARISON TO PRIOR ART
========================

Method                        Energy Reduction   Architecture
HAMT (this work, FF)          15-41%             Any SNN (FF best)
Spike Budgeting (2026)        up to 47%          Global budget
Spike-Thrift (2021)           10-15%             Uniform penalty
Standard spike regularization  5-10%             Uniform penalty

HAMT's advantage: per-neuron learned habituation, discriminates
between novel and familiar inputs, never hurts accuracy.
HAMT's weakness: less effective on convolutional architectures
with natural images.


PENDING: NEUROMORPHIC RESULTS
=============================

The neuromorphic datasets (N-MNIST, SHD, DVS-Gesture) test HAMT on
event-driven data with genuine temporal patterns. These are the
natural habitat for habituation because:

- Events arrive as temporal spike streams (not rate-coded static images)
- Repeated gestures/digits produce predictable temporal patterns
- Habituation should suppress familiar motion patterns efficiently

If neuromorphic results show >20% reduction, the pitch focuses on
edge AI / event-driven applications (keyword detection, gesture
recognition, sensor data processing) where feedforward SNNs and
event cameras dominate. This aligns with the target companies:
BrainChip (Akida), SynSense (Speck), Intel (Loihi).
